# Dossier des captures d'écran

Placez ici les captures d'écran du dashboard en respectant les noms suivants :

| Fichier                     | Section du dashboard                                                        |
| --------------------------- | --------------------------------------------------------------------------- |
| `overview.png`              | Vue d'ensemble — header + KPIs + graphiques principaux                      |
| `exploratory.png`           | Analyse Exploratoire — facteurs de risque (overtime, satisfaction, salaire) |
| `segmentation_pca.png`      | Segmentation — projections PCA 2D/3D avec clusters colorés                  |
| `predictive_models.png`     | Modèles Prédictifs — comparaison des performances + courbes ROC             |
| `individual_prediction.png` | Prédiction Individuelle — formulaire + résultat de risque                   |

**Format recommandé :** PNG, largeur ~1400 px, ratio 16:9
